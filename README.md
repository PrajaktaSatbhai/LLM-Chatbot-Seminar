# LLM Chatbot using Hugging Face & LangChain

This project was developed as part of an academic seminar on **Large Language Models (LLMs)**.
It demonstrates the basic working of an LLM-powered chatbot using **Hugging Face** and **LangChain**.

## Overview
Large Language Models are a class of AI models designed to understand and generate
human-like text. They have shown significant improvements in various NLP tasks.

## Concepts Covered
- Artificial Intelligence & Machine Learning fundamentals
- Large Language Models (LLMs)
- Generative AI
- LLM architecture and working
- Types of LLMs
- Applications of LLMs
- Ethical concerns (bias, privacy, responsible AI)

## Chatbot Implementation
- Built a conversational chatbot using:
  - Hugging Face Transformers
  - LangChain
- Demonstrates prompt handling and response generation

## Applications
- Customer support chatbots
- Educational assistants
- Content generation tools
- Question-answering systems

## Technologies Used
- Python
- Hugging Face
- LangChain
- Jupyter Notebook
